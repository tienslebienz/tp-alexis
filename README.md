Installation
------------

### Cloner le dépot
```sh
$ git clone git@github.com:tienslebienz/tp-alexis.git
```

### Installer les dépendances
```sh
$ yarn
```

### Lancer les test
```sh
# One shot
$ yarn test

# Avec un watch
$ yarn test:watch
```
